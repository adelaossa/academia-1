# Academia Template

Slicing PSD Free Template into a standard HTML and CSS files.

Features:
- HTML Compilant with W3C Validator.
- HTML5 Tags: header, nav, section, article and footer
- HTML Semantic (Working)
- CSS grid made with Zurb Foundation 4
